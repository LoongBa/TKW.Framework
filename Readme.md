# 简单说明
框架定位基于 .NET6/7，主要目的是为团队提供统一的领域级支撑及约束。

# 路线图
## 1. Ver1.0 完成从历史版本到 .NET6 的迁移
### 1.1 迁移并升级代码——目前已基本完成
### 1.2 完整领域级 Demo
## 2. Ver2.0 基于.NET6/7的增强并完善细节，增加扩展的模块、中间件，如 JWT、MQ等
### 2.1 完善 Domain 控制器及领域单元测试
### 2.2 AOP：积累领域过滤器
### 2.3 ORM支持常用数据库

# AdminUI.Scaffold 通用中后台脚手架——目的是直接复制后简单配置即可食用
1. 前端采用vue3.x、AntDesign for Vue，将来支持扩展其它 UIComponent、更换 Theme，甚至配置VI（比如兼容阿里系统设计平台规范）等；
2. 后端采用Net6/7、EF（暂时DBFirst为主）+ OData（UI.Grid扩展，支持低代码Grid）、TKW.Framework.Domain（数据验证默认用 Fluent.Validation）；
3. 菜单管理，权限认证使用 Jwt，支持多终端认证系统；
4. 全局日志、全局异常和错误处理、消息队列（RocketMQ等）、定时任务及任务调度等；
5. 组织架构、字典等常用中后台功能；
6. 多租户、OpenAPI管理（OAuth），支持第三方单点登录 SSO

支持加载动态权限菜单，多方式轻松权限控制
